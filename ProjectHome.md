.net Framework 3.5 class libraries for macro.  for Japanese Only.

マクロ用の .net Framework 3.5 クラスライブラリです。
COM相互運用可能なクラスで、例えば Excel マクロの様な VBA や VBScript などから CreateObject して使うことができます。また、参照設定することも可能です。
